# Online Quiz Maker

A full-stack platform where users can create quizzes, take quizzes one question at a time, and see immediate score feedback.

## Features
- Home page with quick actions
- Quiz creation with multiple-choice questions
- Quiz listing and search
- Quiz taking flow with one-question view
- Results page with correct answers
- User authentication (register/login)
- Mobile responsive UI

## Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB
- Auth: JWT

## Project Structure
- backend: API server and database models
- frontend: React client

## Setup (Development)
1. Backend
   - Create a .env file from .env.example
   - Install dependencies: npm install
   - Start server: npm run dev
   - Seed demo data: npm run seed

2. Frontend
   - Create a .env file from .env.example
   - Install dependencies: npm install
   - Start client: npm run dev
